## Rainbow

### Rainbow.Set
<ParamField path="enabled" type="boolean" required>
    If `enabled` is `true`, rainbow effect will be enabled.
</ParamField>

```lua
ESPLibrary.Rainbow.Set(true)
```

### Rainbow.Enable
Enables the rainbow effect.
```lua
ESPLibrary.Rainbow.Enable()
```

### Rainbow.Disable
Disables the rainbow effect.
```lua
ESPLibrary.Rainbow.Disable()
```

### Rainbow.Toggle
Toggles between the rainbow effect.
```lua
ESPLibrary.Rainbow.Toggle()
```

## Tracers
This exists if you want to have a toggle for Tracers so you don't need to for loop each ESP you created and Update it.

### Tracers.Set
<ParamField path="enabled" type="boolean" required>
    If `enabled` is `true`, tracers will be enabled.
</ParamField>

```lua
ESPLibrary.Tracers.Set(true)
```

### Tracers.Enable
Enables tracers.
```lua
ESPLibrary.Tracers.Enable()
```

### Tracers.Disable
Disables tracers.
```lua
ESPLibrary.Tracers.Disable()
```

### Tracers.Toggle
Toggles between on and off for tracers.
```lua
ESPLibrary.Tracers.Toggle()
```
